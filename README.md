# Litedoge CPU Miner

This is a scrypt web miner with stratum support for LiteDoge mining.

# How to run
* Install any web server to host static files
* Install node.js and npm
* Run ```npm install wsproxy -g```
* Run ```wsproxy -p 8080 &```
